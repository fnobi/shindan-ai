shindan-ai
==============

[![Build Status](https://travis-ci.org/fnobi/shindan-ai.svg?branch=master)](https://travis-ci.org/fnobi/shindan-ai)

shindan-ai client.

## install

### from npm

```
npm install shindan-ai
```

### from github

```
git clone git://github.com/fnobi/shindan-ai.git
```

## cli usage

```
shindan-ai <optionJsonfile>
```
